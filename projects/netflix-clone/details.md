## Overview

A beautifully crafted clone of the Netflix landing page, highlighting streaming app layout conventions and responsive design features.

### Implementation Highlights
- Used semantic HTML and CSS to create the dark, compelling Netflix aesthetic.
- Developed an interactive accordion for the "Frequently Asked Questions" section.
- Integrated high-quality movie banner placeholders.

This project was a great dive into building components intended to handle large background assets while staying accessible and fast across devices.
