## Overview

A fully responsive UI clone of the Amazon Shopping website, focusing on pixel-perfect alignment and core navigational flows using plain HTML and CSS.

### Key Learnings
- Mastered CSS Grid and Flexbox for complex e-commerce layouts.
- Built responsive navigation bars and off-canvas menus.
- Handled product cards positioning mirroring the original Amazon theme.
- Utilized complex CSS selectors for interactive hover states without JavaScript.
